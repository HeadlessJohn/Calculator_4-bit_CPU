# 4-bit CPU Calculator 
## 1. INPUT
### Key input from 4x4 key matrix
```    
     7 8 9 +   
     4 5 6 -   
     1 2 3 x   
     c 0 = /  (c=clear)
```	 
## 2. OUTPUT
### Display operand, operator and calculated numbers to FND controller
```
[operand or operator], [blank], [answer], [answer]
```
